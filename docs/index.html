Sushi Roll

Quickly create layouts using Stylus - chop chop!

Download .zip Download .tar.gz View on GitHub

How it works

Apply the sushi-roll property to any container element. Set your “chop size” (item width…) and combine it with a few special keywords to quickly have your layout done.

If you know CSS flexbox, you’ll quickly understand how this has been implemented. If you don’t know flexbox, that’s fine, because you don’t have too! But it’s really simple and powerful.

Stylus

ul
  sushi-roll: 50px stretch fit 50px

Jade

<ul>
  <li>50px</li>
  <li>stretch</li>
  <li>fit</li>
  <li>50px</li>
</ul>

Output

  • 50px
  • stretch
  • fit
  • 50px

In the above example, you have already designed a header. You can freely switch out the HTML elements, because Sushi Roll doesn’t care about what your elements are, only their order!

This means we can use the exact same Stylus code as above and switch out our HTML to this and have the layout look exactly the same:

<header>
  <img src="logo.png" alt>
  <h2>My lovely article</h2>
  <a href="#"></a>
  <nav id="main-menu">
  </nav>
</header>
